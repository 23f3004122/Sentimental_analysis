Built a two-phase sentiment analysis pipeline using ML (Logistic Regression, TF-IDF) and DL (LSTM with Word2Vec).

Achieved 89% accuracy using L1-regularized Logistic Regression with class_weight='balanced' and K-Fold Cross Validation.

Performed advanced text preprocessing using stopword removal, lemmatization, and regular expressions.

Designed scalable pipelines using scikit-learn and compared feature extraction methods (CountVectorizer, TF-IDF, Word2Vec).

Evaluated and improved model generalization by comparing traditional ML and DL results on informal tweet text.
